Wikipedia-like online encyclopedia

Built with Python/Django/Css/Html

This project has the following features:

1.) Search Encyclopedia 
2.) View all pages 
3.) Create new entry 
4.) Submit entry 
5.) Edit entry's 
6.) Create random page 
7.) Markdown to HTML Conversion
